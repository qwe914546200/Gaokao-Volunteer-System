# 智慧升学平台 — 高考志愿填报模拟系统

面向高考考生的志愿填报辅助模拟工具（非官方，仅供参考）。

## 功能特点
1. 成绩分析：输入各科成绩，自动计算总分并估算全省位次。
2. 冲·稳·保推荐：根据估算位次，智能推荐“冲一冲”、“稳一稳”、“保一保”院校。
3. 院校筛选：支持按院校层次（985/211/双一流等）、地区、类型、批次等多维度筛选。
4. 高校目录：展示全国高校基本信息及排名。
5. 我的志愿表：支持添加、删除、拖拽排序等操作，使用浏览器本地存储。

## 技术栈
- 前端：原生 HTML / CSS / JS，无复杂框架，轻量级，动态交互。
- 后端：Python FastAPI
- 数据库：SQLite

## 本地运行步骤

### 1. 准备环境
确保你的电脑已安装 [Python 3.8+](https://www.python.org/downloads/)。

### 2. 克隆项目
```bash
git clone https://github.com/你的用户名/你的仓库名.git
cd 你的仓库名
```

### 3. 后端服务启动
1. 进入项目根目录。
2. 安装后端依赖：
```bash
pip install -r backend/requirements.txt
```
3. 启动 FastAPI 后端：
```bash
python backend/main.py
```
> 后端服务将在 `http://localhost:8000` 运行。
> 你可以访问 `http://localhost:8000/docs` 查看 API 接口文档。

### 4. 前端访问
前端为纯静态文件，无需额外构建。
直接在浏览器中双击打开 `index.html` 即可使用。
(或者使用任意本地静态服务器打开，如 `python -m http.server 3000`)

## 注意事项
1. 目前数据库中仅包含了**四川省**的录取数据。如果选择其他省份，系统将提示暂无数据。
2. 页面和数据“仅供模拟参考”，不代表官方最终录取结果。
